# bps
A simple app to help with budgeting at reasonable cost

Environment
 - Create a `.env.local` file at the project root with your Supabase values:

```
NEXT_PUBLIC_SUPABASE_URL="https://your-project.supabase.co"
NEXT_PUBLIC_SUPABASE_ANON_KEY="your-anon-key"
```

Next.js will automatically load `.env.local` during development. Do not commit real secrets.
